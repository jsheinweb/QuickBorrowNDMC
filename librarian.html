Power BI Integration Guide
01Export CSV or JSON files using the export buttons above
02Open Power BI Desktop → Get Data → Text/CSV or JSON
03Import Books, Users, Transactions, and Issue Reports as separate tables
04Create relationships: Transactions[UserID]→Users[UserID], Transactions[BookID]→Books[BookID], IssueReports[TransactionID]→Transactions[TransactionID]
05Build visuals: Cards (totals), Line Chart (activity), Bar (top books), Pie (departments), Donut (availability), Table (overdue), Table (issue reports)
06Publish to Power BI Service for shared dashboard access
